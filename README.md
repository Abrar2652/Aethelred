# Project Aethelred

**A Neuro-Causal Framework for Formally Certified Graph Reasoning**

## About this public release

This repository is the **public code release**: our method, experiment and
figure-generation scripts, the numerical results that back every figure, the
camera-ready figures, and the paper source under `docs/paper/`. To keep it
lightweight and license-clean, the following are **not** included and are
regenerated or fetched from upstream:

| Not included | How to obtain |
| --- | --- |
| Datasets (`datasets/<name>/`, `data/`) | `python download_dataset.py` (loaders live in `datasets/*.py`) |
| Trained weights (`checkpoints/`) | regenerated by the `run_*` scripts |
| PGNNCert / SECert baseline (`_ref_pgnncert/`) | official PGNNCert repo (github.com/JetRichardLee/PGNNCert) |
| XGNNCert baseline (`baselines/XGNNCert/`) | official XGNNCert repo (ICLR 2025) |
| DIR upstream (`DIR-GNN-main/`) | official DIR-GNN repo — *note:* a self-contained DIR re-implementation is bundled at `baselines/dir_gnn.py` |
| Run logs, other papers' PDFs | internal only |

The core method (`aethelred_*.py`), the certificates
(`aethelred_edge_certify.py`, `aethelred_pred_certify.py`), and the figure
scripts run without the excluded third-party baselines. The directory tree
below describes the **full research tree** for context.

## How Aethelred Eliminates PGNNCert's T (S Subgraphs)

| PGNNCert | Aethelred | Why |
|---|---|---|
| Hash function h maps edges/nodes to S subgraphs | No hashing. CausalDiscoveryCore learns one causal mask G_c | Learned > static random |
| S independent classifiers f_1...f_S | One unified FocalEngine model | Full graph visible, no fragmentation |
| Majority voting across S outputs | Single forward pass prediction | No accuracy loss from subgraph fragmentation |
| Vote margin P = floor((n_ya - n_yb)/2) | IBP margin: min(salient_low) > max(nonsalient_high) | Certifies reasoning, not just label |
| S is a hyperparameter (tradeoff: more S = more robust but lower accuracy) | No S. num_envs is training-only regularizer | Tradeoff eliminated |
| PGNNCert-E (edge-centric) and PGNNCert-N (node-centric) | Both eliminated. Single causal paradigm | One approach handles all perturbation types |

## Architecture Details

### Three Pillars (aethelred_core.py)

**Pillar 1: CausalDiscoveryCore** — 2-layer GCN that outputs a per-edge mask in [0,1]. Includes IBP forward pass for interval bound propagation.

**Pillar 2: FocalEngine** — Multi-layer GCN with causal mask as edge_weight (gating mechanism). Replaces SharedNodeBackbone.

**Pillar 3: Aethelred** — Unified model. Node classification: direct logits. Graph classification: global_mean_pool then MLP.

### Composite Loss (aethelred_loss.py)

```
L_total = L_task + α·L_invariance + β·L_IB + γ·L_sparsity + δ·L_acyclicity + ε·L_certify
```

- L_task: cross-entropy
- L_invariance: Var(loss across environments) — IRM penalty
- L_IB: sparsity proxy for I(G; G_c)
- L_sparsity: L1 norm on causal mask
- L_acyclicity: NOTEARS Tr(e^{A∘A}) - d
- L_certify: IBP-based, pushes salient/non-salient bounds apart

### Certification (aethelred_certify.py)

Two mechanisms:
1. **IBP**: Propagates [x-ε, x+ε] intervals through CausalDiscoveryCore. Certified if min(salient_lower) > max(nonsalient_upper).
2. **Lipschitz**: Empirically estimates L_g of causal core. Max mask change bounded by L_g · ε.

### Attack Module (aethelred_attacks.py)

Implements PGNNCert's full threat model (Section 2.2):
- `attack_edge_random` — edge injection/deletion (matches {E+, E-})
- `attack_node_injection` — node injection with degree τ (matches Figure 7 protocol)
- `attack_feature_perturbation` — feature poisoning (matches {V_r, E_V_r, X'_V_r})
- `attack_metattack_deeprobust` — **real** Metattack via DeepRobust (ICLR 2019), matches Table 4
- `attack_metattack_approx_deeprobust` — MetaApprox variant for larger graphs
- `attack_metattack` — convenience wrapper (tries DeepRobust, falls back with warning)
- `attack_arbitrary` — combined attack with full PGNNCert budget accounting
- `pyg_to_deeprobust` / `deeprobust_adj_to_pyg_edge_index` — format converters

## Troubleshooting

- **ModuleNotFoundError: torch_geometric** → `pip install torch_geometric`
- **FileNotFoundError: cora_ml.npz** → `python download_dataset.py --dataset Cora-ML`
- **CUDA out of memory on acyclicity** → use `--delta 0.0` to disable
- **Git Bash activate fails** → `source venv/Scripts/activate` or `. venv/Scripts/activate`
