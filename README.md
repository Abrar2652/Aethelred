# Aethelred

**Deterministic, tight certificates for causally-faithful GNN explanations.**

GNN explanations are *fragile*: a small, prediction-preserving perturbation can
rewrite the "reason" behind a decision. A recent line of work *certifies* the
robustness of an explanation — but a certificate of **stability** says nothing
about whether the certified subgraph is the *right* one. A robustly stable
explanation of a spurious correlation is still wrong.

**Aethelred** certifies the robustness of **causally-faithful** explanations. It
couples an invariance-trained, **propagation-free** causal edge scorer with a
**deterministic** edge certificate. Because each edge is scored from its
endpoints alone, its importance is provably invariant to the rest of the graph;
this edge-locality yields a closed-form, per-edge certified radius — **no
voting, no randomized smoothing**, and the certificate is *tight* (a worst-case
adversary achieves exactly the bound). On the state of the art's own metric and
published numbers, Aethelred certifies a **3–6× larger** perturbation size on
real molecular data, stays faithful under a controlled spurious-correlation
sweep where prior explainers collapse, and certifies in a single $O(1)$ pass.

## About this public release

This repository contains our method, the experiment and figure scripts, the
numerical results that back every figure, the camera-ready figures, and the
paper source. To keep it lightweight and license-clean, the following are **not**
included and are regenerated or fetched from upstream:

| Not included | How to obtain |
| --- | --- |
| Datasets (`datasets/<name>/`, `data/`) | `python download_dataset.py` (loaders are in `datasets/*.py`) |
| Trained weights (`checkpoints/`) | regenerated by the `run_*` scripts |
| PGNNCert / SECert baseline (`_ref_pgnncert/`) | official PGNNCert repo (github.com/JetRichardLee/PGNNCert) |
| XGNNCert baseline (`baselines/XGNNCert/`) | official XGNNCert repo (ICLR 2025) |
| DIR upstream (`DIR-GNN-main/`) | official DIR-GNN repo — a self-contained DIR re-implementation is bundled at [`baselines/dir_gnn.py`](baselines/dir_gnn.py) |

The core method, the certificates, and the figure scripts run **without** the
excluded third-party baselines.

```
                             #   edge-locality (F10), tightness (F11), attack faithfulness (F2b)

```

```bash
python -m venv .venv && source .venv/bin/activate

# PyTorch + PyG (CPU shown; for CUDA use the matching index URL)

python download_dataset.py --dataset CiteSeer  # a single dataset
```

```bash
python smoke_test_t56.py
```

## Train

```bash
# graph classification
python run_aethelred_experiment.py --task graph --dataset AIDS --epochs 200

# node classification, with post-training certification
python run_aethelred_experiment.py --task node --dataset Cora-ML --epochs 200 --certify
```

Key flags: `--task {node,graph}`, `--dataset`, `--epochs`, `--lr`, `--certify`,
`--num_envs` (invariance environments), and the loss weights
`--alpha/--beta/--gamma/--delta` (invariance / IB / sparsity / acyclicity).

## Reproduce the paper

**Figures.** The figures in [`figures/`](figures/) are produced by the scripts
below. `figures_phase2.py` rebuilds the comparison figures directly from the
shipped `results/*.json`, so it runs without retraining:

```bash
python figures_phase2.py        # F1, F1b, F2b, F2c, F3, F56, F7, F9
python make_schematic.py        # F0  (architecture schematic)
python make_qual_figures.py     # F4  (qualitative before/after attack)
python make_sensitivity.py      # F8  (hyper-parameter sensitivity)
python make_edge_locality.py    # F10 (edge-locality)
python make_tightness.py        # F11 (certificate tightness)
python make_attack_faith.py     # F2b data (Aethelred under attack)
```

**Tables / experiments.** The full driver writes JSON into `results/`:

```bash
python run_aethelred_comparison.py --quick      # fast sanity pass
python run_aethelred_comparison.py --table all  # all tables
python run_aethelred_comparison.py --figure 7   # a specific figure
python run_aethelred_comparison.py --all        # everything
```

Published SOTA values used as benchmarks (XGNNCert / PGNNCert) are transcribed
and verified in `results/sota_published.json`; the digitized SOTA curves live in
`figures/*_digitized/` with provenance notes.

## Method at a glance

1. **Causal Discovery Core** — a *propagation-free* scorer
   `s(u,v)=σ(MLP[h_u,h_v,|h_u−h_v|,cos(x_u,x_v)])`, `h_u=MLP(x_u)`. An edge's
   score depends only on its endpoints, so it is invariant to every other edge.
2. **Focal Engine** — a GNN whose message passing is gated by the causal mask,
   tying the prediction to the same edges the explanation reports.
3. **Deterministic edge certificate** — edge-locality collapses the usual
   voting/smoothing machinery into a closed-form counting argument: certified
   overlap `|E_k ∩ E'_k| ≥ k − B`, a per-edge radius `R(e_r) = k − r`, and (in
   the transductive setting) provable hijack-impossibility. Proofs are in
   the docstring of [`aethelred_edge_certify.py`](aethelred_edge_certify.py).
